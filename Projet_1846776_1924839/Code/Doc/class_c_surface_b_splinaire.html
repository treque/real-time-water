float CSurfaceBSplinaire::CoxDeBoor ( float  u,
int  i,
int  k,
const float *  Knots 
) [static]

fonction récursive de calculs des coefficients de la B-spline associée

Cette fonction permet de descendre récursivement le long des intervalles adjascents de façon à trouver quels intervalles jouent un rôle (return 1.0f) ou non return (0.0f) dans l'évaluation de la spline courante au point u

Paramètres:
[in] u : la valeur du paramètre d'évaluation (l'endroit sur la spline où on veut évaluer la coordonnée.
[in] i : le numéro de l'intervalle considéré
[in] k : l'ordre courant du polynôme d'évaluation
[in] noeuds : un pointeur sur le vecteur nodal en cours (soit en U.. soit en V)
Renvoie:
1.0 si l'intervalle intervient dans l'évaluation du point considéré 0.0 s'il n'intervient pas.
