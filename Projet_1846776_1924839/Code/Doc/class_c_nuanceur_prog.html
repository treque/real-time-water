constructeur spécifique : utilisé pour créer un PROGRAMME DE NUANCEURS.

global public overloaded CNuanceurProg

Constructeur utilisé pour construire un programme de nuanceurs comportant un nuanceur de sommets et/ou un nuanceur de fragments. Peut produire un programme vide (pour fonctionnalité fixe d'openGL).

Paramètres:
[in] nuanceurSommetsStr char * Nom de fichier du nuanceur de sommets (si = 0 (ou NULL) -> pas de nuanceurs de sommets)
[in] nuanceurFragmentStr char * Nom de fichier du nuanceur de fragments(si = 0 -> pas de nuanceurs de fragments)
[in] compilerMaintenant bool Indique si l'on doit compiler les nuanceurs à la construction de l'objet
Renvoie:
int

void CNuanceurProg::compilerEtLier (  ) 

compile et lie dans openGL les nuanceurs du programme

global public compilerEtLier

Compile et lie les nuanceurs (dans openGL). Vérifie au préalable si les nuanceurs de sommets et de fragments furent définis, car un nuanceur vide (pour la "fixed functionality d'openGL") ne se compile pas.
