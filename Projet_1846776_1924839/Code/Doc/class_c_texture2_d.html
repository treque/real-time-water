Description détaillée

CTexture2D Cette classe encapsule une texture 2D openGL. (type GL_TEXTURE_2D)

Base class CTextureAbstraite <TODO: insérer la description de la classe de base ici>

Remarques:
La classe possède deux constructeurs. Un premier ne prenant qu'un nom de fichier, est utile pour charger des images 2D à partir du disque (BMP uniquement !), et uu deuxième, prenant un vect3D* data, est utile pour créer une texture 2D openGL à partir de données déjà existantes en mémoire. à noter cependant que cette version ne prend QUE des vect3D, i.e. une texture 2D en format GL_FLOAT. Pour d'autres supports de formats, veuillez ajouter les constructeurs appropriés et le code associé.
